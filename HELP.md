### 主题切换
- 点击顶栏 **🌙/☀️** 图标切换亮色/暗色模式
- 点击 **⚙️** 齿轮图标打开设置面板：
  - **背景亮度**：控制背景图片遮罩浓度（0=全透，100=全盖）
  - **页面透明度**：控制气泡玻璃效果（0=全透，100=全实）

## 二、文章管理

### 新建文章
1. 点击左侧栏 **「+ 新建贴文」**
2. 在顶部输入 **路径/标题**（例如 `音乐/日系Trance推荐`）
3. 撰写 Markdown 内容
4. 点击 **「保存」**（首次使用需输入 GitHub Token）

### 编辑已有文章
- 在文章阅读页点击 **「编辑」** 按钮
- 或在文章页 URL 后加 `?edit=路径` 直接打开编辑器

### 文章文件规范
```
Article/{频道}/{年月日-序号-文章名}.md
```

## 三、Markdown 编辑器

### 工具栏

| 按钮 | 快捷键 | 功能 |
|------|--------|------|
| **B** | `Ctrl+B` | 加粗 |
| *I* | `Ctrl+I` | 斜体 |
| **H1** | `Ctrl+1` | 一级标题 |
| **H2** | `Ctrl+2` | 二级标题 |
| 表格图标 | — | 插入表格模板 |
| 引用图标 | — | 插入引用块 |
| 分割线图标 | — | 插入水平分割线 |
| 图片图标 | — | 上传图片并插入 |
| 文件图标 | — | 从仓库选择文件插入 |
| 链接图标 | — | 插入外部下载链接 |
| **分栏预览** | — | 切换编辑/预览分栏模式 |

### Tag 标签系统
- 在编辑器 Tag 输入框中输入文字
- **连续按两次 Enter**（500ms 内）→ 生成 Tag 气泡
- 最多添加 **6 个 Tag**
- 点击 Tag 右侧 **×** 可删除
- Tag 自动保存在文章文件头的 HTML 注释中

### 保存
- 快捷键：`Ctrl+S`
- 首次保存需输入 GitHub Token（会保存在浏览器本地）
- 保存后自动跳转到文章阅读页

## 四、文章阅读

### 页面布局
- **左栏**：频道分类导航（点击筛选文章）
- **中栏**：公告 + 帖子列表（常态）/ 文章内容（浏览态）
- **右栏**：站长信息 / 文章目录（浏览态）

### 代码块
在文章阅览页，代码块顶部显示：
- **左侧**：编程语言名称（如 `JavaScript`、`Python`）
- **右侧**：**「📋 复制」** 按钮（点击一键复制代码内容）

### 图片显示
- 支持 Markdown 图片语法 `![描述](URL)`
- 图片以白色背景显示，自适应宽度
- 可双击图片编辑宽度（百分比 / 像素）

### 文件下载
文章中的文件卡片格式：
```html
<div class="file-card">
  <span>文件名</span>
  <a class="file-download">下载</a>
</div>
```
无需特殊处理，编辑器「文件」和「外部链接」工具自动生成该格式。

## 六、常见问题

### Q: 如何获取 GitHub Token？
1. 访问 https://github.com/settings/tokens
2. 点击 **Generate new token (classic)**
3. 勾选 `repo` 权限
4. 生成并复制 Token
5. 在编辑器第一次保存时输入

### Q: 文章保存在哪里？
文章数据保存在本仓库的 `_legacy/posts/` 目录下，按照 `分类/文件名.md` 结构组织。未来将迁移至 `Article/` 目录。

### Q: 如何更换背景图片？
将图片放入仓库 `Resource/Background/` 目录，命名为 `Default.png`，然后通过设置面板调整背景亮度。

### Q: 如何更换站长头像？
将图片放入仓库 `Resource/Avatar/` 目录，命名为 `Default.png`（格式为 PNG）。
